## 5.1 Identidade de Euler
```
e^(iπ) + 1 = 0
Base dos fasores em circuitos AC
```

## 5.2 Einstein (Relatividade Geral)
```
R_μν - ½g_μν×R = (8πG/c⁴) × T_μν
"Matéria curva o espaço; espaço move a matéria"
```

## 5.3 Gravitação de Newton
```
F = G × (m₁×m₂) / d²
```
