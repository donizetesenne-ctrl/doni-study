# Módulo 4 — Distribuição de Energia

## 4.1 Queda de Tensão
```
ΔV = √3 × I_L × (R×cosφ + X_L×senφ) × ℓ
Limite: ΔV% ≤ 5%
```

## 4.2 Condutores
- Primário: ACSR 2/0 (R=0,153 Ω/km, X=0,306 Ω/km)
- Secundário: Multiplex 2×50mm² + 50mm²(N), I_adm=150A

## 4.3 Aterramento
- R ≤ 10 Ω (haste de cobre)
- Segurança + referência + proteção

## 4.4 Perdas
```
P_cu = 3 × I² × R × ℓ (trifásico)
I dobra → perdas × 4
```
