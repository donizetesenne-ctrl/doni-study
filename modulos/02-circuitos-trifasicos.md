# Módulo 2 — Circuitos Trifásicos

## 2.1 Sistema Trifásico
- 3 fases defasadas 120°, frequência 60 Hz (Brasil)

## 2.2 Tensões
```
V_LL = √3 × V_FN
V_FN = V_LL / √3

Brasil: 13,8kV/7,97kV | 380V/220V | 220V/127V
```

## 2.3 Potências
```
S = √3 × V_LL × I_L      (Aparente, kVA)
P = S × cosφ              (Ativa, kW)
Q = S × senφ              (Reativa, kVAr)
S² = P² + Q²              (Pitágoras!)
```

## 2.4 Correntes
```
I_primário = 300.000 / (1,732 × 13.800) = 12,55 A
I_secundário = 300.000 / (1,732 × 220) = 787 A
```
Menor tensão → maior corrente (mesma potência)
